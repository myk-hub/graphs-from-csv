What is an abnormal point?

Abnormal points are points whose values exceed the the average value of failures per day.
The average value of failures is the average number of failures for 1 day.
For example: we have 27 days and a total of 15 errors. The average value will be 15/27, which equals 0.555555556. Rounding up an integer we get 1. So, all values greater than 1 will be considered as abnormal, you can see this on the chart.
